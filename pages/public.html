Griddify: A Tiny Photoshop Panel to Make Guides and Grids

Griddify is tiny and fast. It helps you compose custom grid systems, vertical rhythm, and do a bunch of other stuff with guides in photoshop.

Watch the short tutorial to learn how it works:

Download and Install (Photoshop CC)

  1. Make sure you have Adobe Extension Manager installed.
  2. Download Griddify, and install using Adobe Extension Manager.
  3. Once installed, open your Photoshop CC. You can find the panel in the “Window” menu, under “Extensions”.

Download and Install (Photoshop CS6)

The CS6 version is now ready! Please note that HTML panels in Photoshop CS6 don't work as well as they do in Photoshop CC, because CC uses an embedded version of Chromium to render HTML content while CS6 uses Flash, which is significantly less stable. So the panel will look and act a bit different than the one we showed you in the demo.

  1. Download the CS6 version of Griddify here.
  2. Extract the zip file.
  3. If you're on windows, put the "Griddify" folder in "C:\Program Files\Adobe\Adobe Photoshop CS6 (64 Bit)\Plug-ins\Panels". If you're on Mac, put the "Griddify" folder in "Macintosh HD/Applications/Adobe Photoshop CS6/Plug-Ins/Panels"
  4. Restart Photoshop CS6, and hopefully you can see the panel in the “Window” menu, under “Extensions”.

We just released CS6, so we do expect it to be less stable for now. If you encounter any issues, please report them on github.

What about CS4/5?

We'll definitely work on a CS4/5 version, but we're not sure if it'll work. You can track the progress here.
