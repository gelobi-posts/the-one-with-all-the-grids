Griddify: A Tiny Photoshop Panel to Make Guides and Grids

Griddify is tiny and fast. It helps you compose custom grid systems, vertical rhythm, and do a bunch of other stuff with guides in photoshop.

Watch the short tutorial to learn how it works:

Download and Install (Photoshop CC)

  1. Make sure you have Adobe Extension Manager installed. You can also get it via Creative Cloud app.
  2. Download Griddify, and install using Adobe Extension Manager.
  3. Once installed, open your Photoshop CC. You can find the panel in the “Window” menu, under “Extensions”.

Download and Install (Photoshop CS6)

  1. Download the CS6 version of Griddify here and extract the zip file.
  2. Put the "Griddify" folder where you installed Photoshop, under "Plugins/Panels"
  3. Restart Photoshop CS6, and hopefully you can see the panel in the “Window” menu, under “Extensions”.

What's new?

Griddify Beta 3 comes with a few bug fixes and improvements, including:
  • A timestamp, which is supposed to fix the "This extension does not contain a valid signature" issue.
  • Better support for larger UI font sizes.
  • No more misplaced guides when using non-pixel ruler units.
  • An actually clickable "Remind Me Later" button for the update notifier.
  • Minor improvements to the UI.
  • And a smarter updater notifier.

Known issues

  • "This functionality may not be available in this version of Photoshop" is an error you might sometimes see. Closing and re-opening the panel seems to fix this error.
  • The panel looks blurry in Retina displays.

Issues regarding Photoshop CS6

HTML panels in Photoshop CS6 don't work as well as they do in Photoshop CC, because CC uses an embedded version of Chromium to render HTML content while CS6 uses Flash, which is significantly less stable. So the panel will look and act a bit differently than the one we showed you in the demo.

We just released CS6, so we do expect it to be less stable for now. If you encounter any issues, please report them on github.

What about CS4/5?

We'll definitely work on a CS4/5 version, but we're not sure if it'll work. You can track the progress here.
