Griddify: A Tiny Photoshop Panel to Make Guides and Grids

Griddify is tiny and fast. It helps you compose custom grid systems, vertical rhythm, and do a bunch of other stuff with guides in photoshop. See the short tutorial to learn how it works:

How to Install

  1. Make sure you have Adobe Extension Manager installed.
  2. Download Griddle, and install using Adobe Extension Manager.
  3. Once installed, open your Photoshop CC. You'll find the panel In the “Window” menu, under “Extensions”.

What about Photoshop CS5/6?

Right now, Griddle only works with Photoshop CC, but surely we can make a version for CS6 and maybe CS5. If you need Griddle for those versions of Photoshop, please let us know (via email or twitter), so that we know we should work on it.

Source Code
